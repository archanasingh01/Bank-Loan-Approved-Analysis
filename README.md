## Bank Loan Approval Analysis

## Project Overview
-This project analyzes bank loan application data to understand applicant characteristics, identify factors influencing loan approvals, and build an interactive 
 Power BI dashboard.
 
## Tools Used
-Microsoft Excel
-Python
-Pandas
-Matplotlib
-SQL (MySQL)
-Power BI

## Project Workflow
-Data Cleaning
-Exploratory Data Analysis (EDA)
-Correlation Analysis
-Business Insights
-Interactive Dashboard

## Key Insights
-Education loans have the highest number of applications.
-Rent and Mortgage applicants account for most loan applications.
-Grade A and Grade B loans are the most common.
-Applicant age is strongly related to credit history length.
-Loan amount is moderately related to loan-to-income ratio.
-Loan approval depends on multiple applicant and financial factors.
